⚙️ Setup & Installation

📋 Requirements
  • Web Server: PHP, Python, Node.js, atau Live Server
  • Browser: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
  • Internet: Untuk load CDN (Bootstrap, PDF.js, Icons)
  • Storage: Minimal 50MB untuk PDF

🚀 Quick Start
1️⃣ PHP 
cd path/to/flippy-optimized
php -S localhost:8000
2️⃣ Python 3 
cd path/to/flippy-optimized
python -m http.server 8000
3️⃣ Node.js 
npm install -g http-server
cd path/to/flippy-optimized
http-server -p 8000
4️⃣ VS Code Live Server 
# 1. Install "Live Server" extension
# 2. Right-click index.html
# 3. Select "Open with Live Server"

Cara Penggunaan

1️⃣ Via HTML Button (Data Attributes)

Cara paling mudah - tambahkan data attributes di button HTML:

<button 
    class="btn btn-primary mode-btn" 
    data-pdf="example/your-file.pdf"
    data-title="My PDF Document"
    data-mode="book"
    data-scale="2.0"
    data-page-width="450"
    data-page-height="600"
    data-sound-enabled="true"
    data-sound-url="dist/sound/turnPage.mp3">
    Buka PDF
</button>
Available Data Attributes:
  • data-pdf - Path ke file PDF (required)
  • data-title - Judul dokumen
  • data-mode - Mode tampilan: 'book', 'webtoon', 'single'
  • data-scale - Kualitas render (1.0 - 3.0)
  • data-page-width - Lebar halaman (px)
  • data-page-height - Tinggi halaman (px)
  • data-sound-enabled - Aktifkan sound: true/false
  • data-sound-url - Path ke file MP3
2️⃣ Via JavaScript (Full Options)

Untuk kontrol penuh, gunakan JavaScript:

const flippy = new Flippy({
    // Required
    pdfUrl: 'example/your-file.pdf',

    // Display Settings
    mode: 'book',              // 'book' | 'webtoon' | 'single'
    title: 'My PDF Document',  // Document title
    
    // Rendering Quality
    scale: 2.0,                // 1.0 - 3.0 (default: 2.0)
    jpegQuality: 0.92,         // 0.0 - 1.0 (default: 0.92)
    
    // Book Mode Dimensions
    pageWidth: 450,            // Page width in pixels
    pageHeight: 600,           // Page height in pixels
    
    // Zoom Controls
    minZoom: 0.5,              // Minimum zoom level
    maxZoom: 3.0,              // Maximum zoom level
    zoomStep: 0.1,             // Zoom increment per scroll
    
    // Performance
    parallelRender: 2,         // Pages to render in parallel (1-4)
    
    // Sound Effects
    soundEnabled: true,        // Enable page flip sound
    soundUrl: 'dist/sound/turnPage.mp3'
});

// Open the viewer
flippy.open();
All Available Options:
  • pdfUrl (string, required) - Path ke file PDF
  • mode (string) - Mode: 'book' | 'webtoon' | 'single' (default: 'book')
  • title (string) - Judul di header (default: 'PDF Viewer')
  • scale (number) - Render quality 1.0-3.0 (default: 2.0)
  • jpegQuality (number) - JPEG quality 0.0-1.0 (default: 0.92)
  • pageWidth (number) - Lebar halaman px (default: 450)
  • pageHeight (number) - Tinggi halaman px (default: 600)
  • minZoom (number) - Min zoom (default: 0.5)
  • maxZoom (number) - Max zoom (default: 3.0)
  • zoomStep (number) - Zoom increment (default: 0.1)
  • parallelRender (number) - Parallel pages 1-4 (default: 2)
  • soundEnabled (boolean) - Sound on/off (default: true)
  • soundUrl (string) - Path ke MP3 (default: 'dist/sound/turnPage.mp3')
3️⃣ Mode Options
  • book - 3D flip effect
  • webtoon - Vertical scroll
  • single - Horizontal slide
4️⃣ Keyboard Shortcuts
  • ← / → - Navigate pages
  • Home / End - First/Last page
  • Scroll - Zoom
  • Esc - Close

        <!-- Method 1: HTML Button -->

                data-pdf="example/sample.pdf"
                data-mode="book">
            Open with HTML Button
        </button>
        
        <!-- Method 2: JavaScript -->
        <button class="btn btn-success" onclick="openWithJS()">
            Open with JavaScript
        </button>
    </div>

    <!-- Bootstrap JS -->
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>

    <script src="https://cdn.jsdelivr.net/npm/pdfjs-dist@3.11.174/build/pdf.min.js"></script>
    <script>
        if (typeof pdfjsLib !== 'undefined') {
            pdfjsLib.GlobalWorkerOptions.workerSrc = 
                'https://cdn.jsdelivr.net/npm/pdfjs-dist@3.11.174/build/pdf.worker.min.js';
        }
    </script>
    <!-- Flippy JS -->
    <script src="dist/js/flippy.min.js"></script>

    <script>
        // Method 2: JavaScript Function
        function openWithJS() {
            const flippy = new Flippy({
                pdfUrl: 'example/sample.pdf',
                mode: 'webtoon',
                title: 'My Custom PDF',
                scale: 1.8,
                soundEnabled: false
            });
            flippy.open();
        }

        // Initialize HTML buttons
        document.addEventListener('DOMContentLoaded', function() {
            document.querySelectorAll('.mode-btn').forEach(btn => {
                btn.addEventListener('click', function() {
                    const flippy = new Flippy({
                        pdfUrl: this.dataset.pdf,
                        mode: this.dataset.mode || 'book',
                        title: this.dataset.title || 'PDF Viewer'
                    });
                    flippy.open();
                });
            });
        });
    </script>
</body>
</html>
