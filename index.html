Cara Penggunaan

1Ô∏è‚É£ Init .mode-btn (Wajib jika pakai tombol)

Kalau kamu pakai class .mode-btn, tambahkan inisiasi JS ini supaya tombolnya ter-bind & Flippy kebuka saat diklik:

document.addEventListener('DOMContentLoaded', () => {
  // Pastikan PDF.js & Flippy sudah ada
  const ready = setInterval(() => {
    if (typeof pdfjsLib !== 'undefined' && typeof Flippy !== 'undefined') {
      clearInterval(ready);
      document.querySelectorAll('.mode-btn').forEach(btn => {
        btn.disabled = false;
        btn.addEventListener('click', () => {
          const pdf = btn.dataset.pdf;
          if (!pdf || !pdf.endsWith('.pdf')) return alert('‚ùå Invalid PDF file path');
          const flippy = new Flippy({
            pdfUrl: pdf,
            title: btn.dataset.title || 'PDF Viewer',
            mode: btn.dataset.mode || 'book',
            soundEnabled: true
          });
          flippy.open();
        });
      });
    }
  }, 200);
});
2Ô∏è‚É£ Via HTML Button (Data Attributes)

Cara paling mudah - tambahkan data attributes di button HTML:

<button 
  class="btn btn-primary mode-btn" 
  data-pdf="example/your-file.pdf"
  data-title="My PDF Document"
  data-mode="book"
  data-scale="2.0"
  data-page-width="450"
  data-page-height="600"
  data-sound-enabled="true"
  data-sound-url="dist/sound/turnPage.mp3">Buka PDF</button>
Available Data Attributes:
  • data-pdf - Path ke file PDF (required)
  • data-title - Judul dokumen
  • data-mode - Mode tampilan: 'book', 'webtoon', 'single'
  • data-scale - Kualitas render (1.0 - 3.0)
  • data-page-width - Lebar halaman (px)
  • data-page-height - Tinggi halaman (px)
  • data-sound-enabled - Aktifkan sound: true/false
  • data-sound-url - Path ke file MP3
3Ô∏è‚É£ Via JavaScript (Full Options)

Untuk kontrol penuh, gunakan JavaScript:

const flippy = new Flippy({
  // Required
  pdfUrl: 'example/your-file.pdf',

  // Display Settings
  mode: 'book',              // 'book' | 'webtoon' | 'single'
  title: 'My PDF Document',  // Document title
  
  // Rendering Quality
  scale: 2.0,                // 1.0 - 3.0 (default: 2.0)
  jpegQuality: 0.92,         // 0.0 - 1.0 (default: 0.92)
  
  // Book Mode Dimensions
  pageWidth: 450,            // Page width in pixels
  pageHeight: 600,           // Page height in pixels
  
  // Zoom Controls
  minZoom: 0.5,              // Minimum zoom level
  maxZoom: 3.0,              // Maximum zoom level
  zoomStep: 0.1,             // Zoom increment per scroll
  
  // Performance
  parallelRender: 2,         // Pages to render in parallel (1-4)
  
  // Sound Effects
  soundEnabled: true,        // Enable page flip sound
  soundUrl: 'dist/sound/turnPage.mp3'
});
flippy.open();
All Available Options:
  • pdfUrl (string, required) - Path ke file PDF
  • mode (string) - Mode: 'book' | 'webtoon' | 'single' (default: 'book')
  • title (string) - Judul di header (default: 'PDF Viewer')
  • scale (number) - Render quality 1.0-3.0 (default: 2.0)
  • jpegQuality (number) - JPEG quality 0.0-1.0 (default: 0.92)
  • pageWidth (number) - Lebar halaman px (default: 450)
  • pageHeight (number) - Tinggi halaman px (default: 600)
  • minZoom (number) - Min zoom (default: 0.5)
  • maxZoom (number) - Max zoom (default: 3.0)
  • zoomStep (number) - Zoom increment (default: 0.1)
  • parallelRender (number) - Parallel pages 1-4 (default: 2)
  • soundEnabled (boolean) - Sound on/off (default: true)
  • soundUrl (string) - Path ke MP3 (default: 'dist/sound/turnPage.mp3')
